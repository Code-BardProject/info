# Анкета для педагогов

## Описание проекта

Интерактивная веб-форма для сбора информации о педагогических работниках. Форма оформлена в современном стиле с анимациями и эффектами.

## Функциональность

### Поля формы:
1. **Уровень образования** — выбор между средним, средним специальным, высшим, или вариант "учусь"
2. **Учебное заведение** — университет, колледж, техникум или другое
3. **Место работы** — институт, колледж, техникум, школа или другое
4. **Предмет** — математика, языки, физика, история, химия, биология и др.
5. **Стаж работы** — от 1-2 лет до 5+ лет
6. **Классы** — начальный, 2-4, 5-9, 10-11 классы
7. **Должность** — директор, зам. директор, завуч, учитель, учительница, психолог
8. **Дополнительная информация** — текстовое поле для свободного ввода

### Особенности:
- Поле "другое" в каждом селекте открывает текстовый input для ручного ввода
- Плавные анимации появления элементов
- Стеклянный эффект (glassmorphism) для блоков
- Адаптивный дизайн

## Технологии

- HTML5
- CSS3 (анимации, градиенты, backdrop-filter)
- JavaScript (интерактивность)

## Визуальные эффекты

- **Фон:** фиолетовый градиент (135°)
- **Блоки:** полупрозрачный белый фон с blur-эффектом
- **Навигация:** фиксированное меню с glassmorphism эффектом
- **Анимации:** 
  - Последовательное появление блоков (0.1s — 0.8s задержка)
  - Подъём при наведении на селекты
  - Glow-эффект при фокусе
  - Плавное появление input полей
  - Появление меню сверху (slide down)
  - Hover-эффекты на логотип и ссылки
